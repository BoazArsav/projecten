Programmeren

De reden dat je deze opleiding gekozen heb.


Programmeren is het schrijven van een computerprogramma, een concrete reeks instructies die een computer kan uitvoeren. Dit is de taak van een softwareontwikkelaar of programmeur(dus jij). Programmeren wordt in het algemeen niet direct in machinetaal gedaan, maar in een programmeertaal. Er zijn heel veel programeer talen (teveel om op te noemen)

Wij hebben tot nu toe alleen maar HTML en CSS gekregen, maar we krijgen mogenlijk ook deze talen:
PHP
MySQL
Javascript
Python
C# en andere, maar die vallen dan onder keuzevakken.

De lessen programmeren bestaat uit meerdere delen: Project Managment, Programmeren, Project en Ontwerpen. Deze lessen zijn er de hele week door en worden meerendeels gegeven door Pete Berkenhout, maar alleen op de dinsdag hebben wij les van meneer van Schaik.

Dit is gewoon je opleiding, als je dit niet kan, dan haal jij je examens gewoon niet.

Project

Een meer gedetailleerde versie van de uitleg over deze website!


Deze website is gemaakt op opdracht van meneer Pete Berkenhout om ons te laten leren hoe we moeten werken met HTML en CSS in de praktijk, hiermee kunnen we aantonen wat wij geleerd hebben aan de leraren en ouders.

Wij hebben voor onze website eisen gekregen die we moeten volgen, deze zijn:
De website moet uit minimaal 10 pagina's bestaan.
Het moet een navigatiebalk hebben.
Het moet een formulier hebben.
De site moet een tabel hebben (gebruik het voor vaken of de examenplan).
Alle content (alles wat erin staat) moet zelfgemaakt zijn.

We moeten de volgende onderwerpen in de website bespreken:
Een homepage
Learning Matters
Eduarte en de app
De opleidingwijzer
Locatiegids
SLB
Beschikbare software (office 365, Visual Studio Code en XAMPP)
Opbouw opleiding
Keuzedelen
Tabel met vakken
Examenplan
VGT/POM
BSA
Ouderavonden (datums en doelen van)
Over ons (studenten)
Fotogallerij
Een Beoordelingsformulier (voor de ouders)

De project wordt gemaakt in groepen van 2 (sommigen zijn meer), en je moet je project regelmatig uploaden en updaten via Github zodat je partners dezelfde documenten hebben die ze dan ook kunnen controleren en bewerken.

De project (deze website) moet ook gevalideerd worden op code fouten (HTML en CSS), zodat je kan zien dat er mogenlijk problemen gaan onstaan.

We moeten ook bij de ouderavond een Powerpoint-Presentatie geven, die gaat dan over 1 onderwerp, in plaats van meerdere.

De deadline van het project is dat je het 1 week na de ouderavond inlevert, maar de website en powerpoint moet je op 13 oktober kunnen tonen al.

Je krijgt voor de website een cijfer, maar de powerpoint telt niet mee, die is alleen extra punten.
