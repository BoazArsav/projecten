Visual Studio Code

Onze werkomgeving!


Visual Studio Code is een code editor die gemaakt is door Microsoft, hiermee kunnen wij als studenten coderen en programmeren in een snelle en makkelijke aanpasbare ontwikkelingsruimte.

Visual Studio Code is makkelijk aanpasbaar, omdat je allemaal verschillende plugin's kan installeren door op een knop te klikken en dan de proggramma te zoeken die jij nodig heb.

Visual Studio Code heeft ook Git intregratie, hiermee kunnen wij al onze code en bestanden gelijk doorzenden naar een server, zodat je altijd de meest recente versie op het internet heb staan die mogenlijk dan ook door je partners bewerkt kan worden.

Visual Studio Code is gratis voor iedereen om te gebruiken en is open-source, dus iedereen kan de software bewerken zo nodig.

XAMPP

Onze (Lokale) Serverclient


XAMPP is een Open-Source project gemaakt door ApacheFriends, de software bestaat uit meerdere dingen, zoals:

Een Apache-Webserver
MariaDB
PHP
Perl

XAMPP is uitgekomen in 2002 en was orgineel 2 programma's, namelijk WAMMP voor Windows en LAMMP voor Linux.

Wij gebruiken XAMPP zodat wij onze website's en programma's op een lokale server kunnen testen zoals het op het internet zou zijn.
