Git

De oorsprong van Github/Gitlab


Git is een VersieBeheerSysteem (VBS), dit houd in dat het een programma is waarmee mensen de weizigingen in hun documenten en programma's kunnen bijhouden en doorzenden naar een server (de cloud)

Git is gemaakt met snelheid in gedachte, dat betekent dat dit programma heel responsief moet zijn van nature, ook houd dit in dat de programma zelf niet al te groot moet zijn.

Github en Gitlab bestaan uit code dat van oorsprong van Git zelf is, dus de programma's hebben veel dezelfde functie's die van Git komen, met lichte variaties.

Github

Waar wij onze projecten opslaan


Github is de software (of website) die wij gebruiken om onze projecten online op te slaan in repository's. Github is een uitgebeide versie van git, dat betekent dat Github meer functies heeft dan Git alleen.

Github wordt door onz gebruikt om projecten te uploaden, hierdoor staat er altijd op het internet een versie van onze project waardoor je nooit de smoes heb dat jij je werk verwijderd heb, want er staat een kopie permanent op het internet.

In Github werk je met repository's, dit betekent dat al je projecten hun eigen online map hebben waardoor alles gesorteerd is en makkelijker gearchiveerd wordt, ook is het mogenlijk om meerdere mensen in een repository te laten werken, daarmee kan je in projecten samenwerken en waardoor mensen gewoon hun nieuwste bestanden kunnen pushen naar de repository en de rest gelijk de nieuwste zelf ook hebben.

Gitlab

Wat wij (nog niet) gebruiken


Ik kan niet veel zeggen over Gitlab, omdat ik nog geen ervaring ermee heb, dus info kan fout zijn.

Gitlab is de Open-Source versie van Github, dit houd in dat je voor Gitlab je eigen code kan schrijven en voor Github weer niet. Ook is Gitlab geschreven in andere codeertaal(en) dan Github, dat houd in dat als je code wil schrijven voor Gitlab, dat het ook in de correcte programeertaal is.

Gitlab is meer gefocust op grotere programma's, bij Gitlab is er geen bestandlimit en bij Github wel, zolang je project maximaal 10GB is, kan je het op Gitlab posten.

Linus Torvalds.


Linus Torvalds is de maker van Git (en dus eigenlijk ook de reden van Github en Gitlab's bestaan), hij is ook de maker van de OS Linux, de (BETERE) alternatief voor Windows.
